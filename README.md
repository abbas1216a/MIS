# School Management System (SMS)

A simple School Management System built using HTML, CSS, and JavaScript.

## Project Structure

```text
sms/
├── index.html
├── dashboard.html
├── register.html
├── README.md
├── .nojekyll
├── css/
│   ├── style.css
│   └── dashboard.css
└── js/
    ├── auth.js
    └── dashboard.js
```

## Features

* User Registration
* Login Authentication
* Dashboard Management
* Responsive Design
* Built with HTML, CSS, and JavaScript

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Deployment

This project is deployed using GitHub Pages through a GitHub Actions workflow (`pages.yml`).

After pushing changes to the repository, GitHub Actions automatically builds and deploys the website.
